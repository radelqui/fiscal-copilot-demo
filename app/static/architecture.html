Componentes

Auth + Rate Limiting

Token de 14 días, 30 req/hora, cap $2/día.
app/auth.py

Bedrock Agent

Agent 2BOPZRAI7X con Sonnet 4.6. enableTrace extrae tokens reales de cada paso.
app/bedrock_agent.py

Knowledge Base (RAG)

KB 5I5RDNA2V1 con S3 Vectors. Titan V2 1024d. Corpus fiscal DGII con source attribution.
Corpus local: app/rag/corpus/

Guardrail

Guardrail xgn38kcg6hrq: bloquea prompt injection, temas off-topic, PII.
Test: evals/golden_set.jsonl caso injection-001

Tools (Action Group)

Lambda fiscal-copilot-tools: calcular_itbis, validar_ncf, presentar_formato_606.
aws/lambda/handler.py · app/tools/

HITL (Human-in-the-Loop)

presentar_formato_606 con requireConfirmation. returnControl → bandeja → aprobar/rechazar.
specs/hitl-workflow.md · app/routers/demo.py

Observabilidad

PostgreSQL persiste cada traza: tokens, coste, latencia, tools, por tenant y modelo.
/metrics (JSON p50/p95) · /dashboard (HTML)

Evals Pipeline

8 casos golden set. 4 métricas LLM-as-Judge (Haiku). Router 4 modelos. Coste $0.018.
evals/ · reports/comparativa.md
